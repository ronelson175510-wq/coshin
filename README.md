# Coshin.com

A global social media platform built with Flask backend and vanilla JavaScript frontend.

## 🌍 Live Demo

- **Frontend:** https://Coshin.vercel.app (update after deployment)
- **Backend API:** https://Coshin-backend.onrender.com (update after deployment)

## 🚀 Quick Start

See [QUICK_DEPLOY.md](QUICK_DEPLOY.md) for deployment instructions.

## 📁 Project Structure

```
Coshin.com/
├── backend/           # Flask API server
│   ├── app.py        # Main application
│   ├── requirements.txt
│   └── .env.example  # Environment template
├── *.html            # Frontend pages
├── coshin-api.js        # API client
└── style.css         # Styles
```

## 🛠️ Tech Stack

- **Backend:** Flask, MySQL, JWT Auth
- **Frontend:** Vanilla JS, HTML5, CSS3
- **Hosting:** Render (backend), Vercel (frontend)
- **Database:** MySQL (PlanetScale/Railway)

See `backend/.env.example` for required variables.
