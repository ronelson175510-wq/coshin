Help

How to configure the API server?

  1. Enter your API server URL above
  2. Click "Test Connection" to verify it works
  3. Click "Save Settings" to store your preference

Testing with ngrok?

  1. Download ngrok from ngrok.com
  2. Run: ngrok http 5000
  3. Copy the HTTPS URL from ngrok output
  4. Paste it above and save

Deploying to production?

  1. See DEPLOYMENT_GUIDE.md for full instructions
  2. Update this setting to your cloud backend URL
  3. All users will use your production server
